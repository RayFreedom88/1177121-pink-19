#### При работе над данным проектом был использован следующий стек технологии:

- HTML, CSS, JavaScript;
- семантическая и валидная разметка;
- Git и GitHub;
- Adobe Photoshop;
- формы;
- доступность всего контента и интерфейса;
- сетки на Flex-box'ах;
- методология БЭМ
- препроцессор Sass;
- сборщик Gulp;
- адаптивная верстка;
- адаптивная графика;
- Svg;
- ретинизация графики;
- Pixel Perfect;
- mobile first.
